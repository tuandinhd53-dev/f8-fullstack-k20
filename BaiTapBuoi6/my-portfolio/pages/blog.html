Nội dung

Giới thiệu

Frontend Development là lĩnh vực tôi rất yêu thích trong quá trình học lập trình web.

Tôi bắt đầu với HTML và CSS để xây dựng giao diện website cơ bản.

Điều thú vị nhất là khả năng biến ý tưởng thành giao diện thực tế bằng code.

Học HTML & CSS

Semantic HTML giúp website dễ đọc và tối ưu SEO hơn.

Tôi thường sử dụng Flexbox và Grid để xây dựng layout hiện đại.

Một số thẻ quan trọng như section, article và nav giúp cấu trúc website rõ ràng hơn.

Sau khi học HTML và CSS, tôi bắt đầu tìm hiểu JavaScript

JavaScript giúp website trở nên tương tác hơn thông qua xử lý DOM và events.

Tôi thường dùng Ctrl + S để lưu code nhanh trong VS Code.
